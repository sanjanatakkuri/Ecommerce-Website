Ecommerce Website using Flask & MySQL

A full-stack e-commerce web application built using **Flask** and **MySQL**, supporting user authentication, product management, shopping cart functionality, and email OTP verification.


 User Module
- User registration with email OTP verification
- Secure user login
- Browse available products
- Add products to cart
- View and manage shopping cart
- Checkout functionality

 Admin Module
- Admin login
- Add new products with image upload
- View all products
- Delete products
- Manage product stock

 Product Management
- Product image storage and display
- Stock quantity handling
- Discounted price support

Technologies Used

- Backend:Python, Flask
- Frontend: HTML, CSS
- Database: MySQL
- Email Service:SMTP (Gmail)
- Database Connector: PyMySQL

Database Structure

- USERS – stores user details
- PRODUCTS– stores product information
- CART – stores user cart items
